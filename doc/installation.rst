Installation
============

.. toctree::
   :maxdepth: 2
   :caption: Contents:

To install:

.. code-block:: python

  python3 -m venv
  . ./venv/bin/activate
  pip install miros


Miros has been tested on Python 3.5 and is only dependent upon the Python
standard library.
